# PANDIX
Pandix is a **fictional** operating system created by some guy just for fun, this is not intented to be a "real OS" (cause it isn't), it is just a small, personal project for fun.

# Commands
Here an updated list of commands ts has:  
`exit` and `e`: Leave the program  
`version`: See the version of the program  
`fetch`: See the system's details (don't expect actual details)
